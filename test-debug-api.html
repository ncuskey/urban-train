🔧 Console Debug Tools Available

After the page loads, you can use these commands in the DevTools console:

  • LabelsDebug.step0() - Clear placement state (bump epoch, clear DOM, reset store)
  • LabelsDebug.epoch() - Get current placement epoch
  • LabelsDebug.bumpEpoch() - Increment placement epoch
  • LabelsDebug.cancel() - Cancel any pending placement work
